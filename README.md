# Webpack4 Boilerplate
Webpack Frontend Boilerplate with Nunjucks, Sprites, SASS and Babel.

## Prerequisite
Node 10.11.0

## Install Setup
Load node.js version
```
nvm use
```

Install Packages
```
npm install
```

## Run setup (prod / dev)
Run watcher
```
npm run dev
```

Run production
```
npm run prod
```
