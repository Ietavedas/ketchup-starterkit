# Webpack4 Boilerplate
Starterkit for frontend tasks with Nunjucks templates, ES201X transpiler and Sass precompiling with SVG/PNG sprites.

## Prerequisite
Node 10.11.0

## Install Setup
Load node.js version
```
nvm use
```

Install Packages
```
npm install
```

## Run setup (prod / dev)
Run watcher
```
npm run dev
```

Run production
```
npm run prod
```
