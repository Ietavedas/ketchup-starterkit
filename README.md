# Webpack4 Boilerplate
No-Gulp Boilerplate with ES6 Transpiler, SMACCS based Sass and Sprite Generator (PNG/SVG).
Hot load with Browsersync.

## Prerequisite
Node 8.9.4

## Install Setup
Load Node.js version
```
nvm use
```

Install Packages
```
npm install
```

## Run setup (prod / dev)
Run watcher
```
cd site/web/themes/unic/
npm run dev
```

Run production
```
cd site/web/themes/unic/
npm run prod
```
